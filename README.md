# Find Cat

**Find Cat** — казуальная hidden-object игра на **Unity**, в которой игрок ищет кота на изображении, проходит уровни, получает монеты за успешные находки, использует подсказки и может пропускать уровни за просмотр rewarded-рекламы.

Проект ориентирован на **WebGL / Yandex Games**: в нём есть сохранения, таблица лидеров, полноэкранная реклама, rewarded-реклама и простая структура сцены, рассчитанная на короткие игровые сессии.

## Что есть в проекте

* одна основная игровая сцена;
* прогрессия уровней с сохранением прогресса;
* внутриигровая валюта;
* система подсказок;
* система жизней и ошибок;
* таймер текущего уровня;
* пропуск уровня за rewarded-рекламу;
* отправка результата в таблицу лидеров через Yandex Games;
* отдельное хранение координат кота в JSON.

## Игровой цикл

1. Загружается изображение уровня.
2. Игрок ищет кота на картинке.
3. При успешном клике происходит переход на следующий уровень.
4. За прохождение игрок получает монеты.
5. Подсказки покупаются за монеты.
6. При слишком большом количестве ошибок жизни восстанавливаются, а внутриигровая валюта уменьшается.
7. Уровень можно пропустить через rewarded-рекламу.

## Основные скрипты

### `Levels.cs`

Главный контроллер игрового процесса:

* загружает изображение текущего уровня;
* хранит текущий уровень, цикл прохождения и количество монет;
* переводит игрока на следующий уровень после успешного попадания;
* сохраняет прогресс;
* отправляет результат в лидерборд;
* вызывает рекламу.

### `DevCatSet.cs`

Вспомогательный инструмент для работы с координатами кота:

* загружает координаты из `Assets/StreamingAssets/CatPos.json`;
* выставляет позицию кота для текущего уровня;
* позволяет сохранять обновлённые координаты в редакторе.

### `Hearts.cs`

Отвечает за жизни, обработку ошибок и восстановление после проигрыша.

### `HelpPanel.cs`

Управляет панелью подсказки и логикой её покупки.

### `Timer.cs`

Считает время прохождения текущего уровня.

### `Loose.cs`

Управляет затемнением экрана и UI-переходами в состоянии проигрыша.

## Данные уровней

Координаты кота хранятся в файле:

```text
Assets/StreamingAssets/CatPos.json
```

В этом файле лежат заранее подготовленные координаты для уровней. В редакторе их можно править через логику, реализованную в `DevCatSet.cs`.

## Внешняя зависимость от контента

Изображения уровней не лежат напрямую в репозитории. Они подгружаются во время игры из внешнего источника. Поэтому:

* для нормальной работы нужен доступ в интернет;
* удалённый источник изображений должен быть доступен;
* для полностью офлайн-тестирования уровней нужно либо сохранять изображения локально, либо менять логику загрузки.

## Сохранения и платформенная интеграция

Проект использует **Yandex Games** для:

* загрузки и сохранения прогресса;
* обновления таблицы лидеров;
* rewarded-рекламы;
* полноэкранной рекламы.

В сохранениях используются следующие игровые данные:

* текущий уровень;
* количество монет;
* счётчик циклов после прохождения полного набора уровней.

## Запуск проекта

### Требования

* Unity **2022.3.34f1**

### Локальный запуск

1. Открыть Unity Hub.
2. Добавить папку проекта.
3. Открыть проект в Unity **2022.3.34f1**.
4. Открыть сцену:

```text
Assets/_Scenes/GameScene.unity
```

5. Запустить сцену в Play Mode.
